## `zane.fun`
Repository hosting the code for my webpage [zane.fun](https://zane.fun)

#### Development

To set up your environment

```bash
git clone https://github.com/zkharit/zane.fun
cd zane.fun
npm install
```

To run the development server

```bash
npm run dev
```

#### Deployment

To deploy

```bash
npm run build
npm run start
```
